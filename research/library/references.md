# References — Threat Catalog (SNH Interventions for Remote-Worker Communities)

**What this is.** A consolidated, de-duplicated bibliography of every distinct source cited in the
threat catalog. Each source appears **once**, with (where the document supplies it) authors, year,
title, venue, identifier (DOI / PMC / PMID / arXiv / URL), a brief scale/context note, and the
threat IDs that cite it.

**Source document.** `../foundational/threat_catalog.md` (this repo; originally drafted in the
`host-happily-hub` repo as v1.1, 2026-06-09; main catalog A1–K6 + Completeness Addendum §§1–6).

**Compiled.** 2026-06-10.

**Numbering convention — IMPORTANT.** References are numbered `[1]`, `[2]`, … in **order of first
appearance** in the source document (NOT alphabetical). This is an **append-only** scheme: a new
source gets the next free number and existing numbers never shift, so any document that cites `[n]`
stays valid forever. New references discovered later are appended at the end with the next number.

**Identifiers.** Only identifiers actually present in the source document are reproduced here.
Where the document gives no DOI/PMC/PMID/arXiv/URL, the identifier field is left blank and the entry
is also listed under **"Needs a DOI / identifier."** No identifiers were fabricated.

**"Cited by" / threat IDs.** Inferred from proximity in the document (the threat entry, addendum
section, quarantine item, or success-lead under which the source appears). Sources used in the
"success-model leads" (Addendum §5) are tagged `success-lead` plus the threat IDs they are said to
mitigate.
